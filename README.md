# nanoCluster
An edge compute platform

<img src="/Documentation/Images/top.jpg" alt="nanoCluster">

This is a simple cluster, composed of four NVIDIA Jetson Nano single board computers (providing the cluster's essentail computational resources) together with an AdaFruit Grand Central (serving a the cluster's monitoring and control panel).

Slack channel for this project resides here: https://booch-nanocluster.slack.com
